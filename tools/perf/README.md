# Performance Budgets Module

This directory contains the opt-in performance budgets module for Atlas.

## Contents

- `enable.js` - Script to enable performance workflows
- `disable.js` - Script to disable performance workflows
- `templates/` - GitHub Actions workflow templates

## Usage

Enable performance checks:

```bash
pnpm perf:enable
```

Disable performance checks:

```bash
pnpm perf:disable
```
