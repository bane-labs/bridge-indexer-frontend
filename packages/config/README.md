# @atlas/config

Shared configuration packages for the frontend platform monorepo.

## Exports

- `@atlas/config/eslint` - ESLint configuration
- `@atlas/config/typescript` - Base TypeScript configuration
- `@atlas/config/typescript/nextjs` - Next.js TypeScript configuration
- `@atlas/config/typescript/react-library` - React library TypeScript configuration
- `@atlas/config/prettier` - Prettier configuration
- `@atlas/config/jest` - Jest configuration for React components

## Usage

Install as a workspace dependency and extend configurations in your app/package.
