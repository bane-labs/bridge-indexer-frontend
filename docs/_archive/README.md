# Archived Documentation

This directory contains documentation that has been superseded by the canonical docs in
`docs/how-we-build/`.

## Why Archive?

Rather than delete historical documentation, we preserve it here for reference. This allows:

- Tracing the evolution of platform decisions
- Understanding context behind current patterns
- Recovering useful information if needed

## Archive Contents

### 2025-12-pre-platform-docs/

Documentation that existed before the December 2025 documentation reset. These files contain
valuable historical context but may be outdated, duplicated, or inconsistent with current
implementation.

**Status**: Superseded by `docs/how-we-build/` and `docs/adr/`

## Rules

1. **Do not modify archived content** — preserve original state
2. **Do not reference archived docs as authoritative** — use `docs/how-we-build/` instead
3. **New experimental docs go here** — until they're proven and promoted to canonical status
